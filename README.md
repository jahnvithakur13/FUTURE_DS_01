# FUTURE_DS_01
Sales data analysis using Python (Pandas, Matplotlib, Seaborn)

📌 Project Overview:
This project focuses on analyzing a retail sales dataset to understand business performance in terms of revenue, profit, and product trends. The aim is to extract meaningful insights from the data and provide practical recommendations that can help improve decision-making and overall profitability.

🎯 Objectives:
The main objectives of this project are to clean and preprocess the raw sales data, analyze revenue trends over time, identify top-performing products and categories, evaluate regional performance, study the impact of discounts on profit, and generate actionable business insights.

🛠️ Tools Used:
This project is implemented using Python along with libraries such as Pandas for data manipulation, Matplotlib and Seaborn for data visualization, and Jupyter Notebook for analysis and presentation.

📂 Dataset:
* Superstore Sales Dataset (CSV format)
* Contains:
* Order details
* Customer information
* Product categories and sub-categories
* Sales, profit, and discount data

🔍 Analysis Performed:
The analysis includes data cleaning and preparation, calculation of key business metrics like total sales and profit, time-based analysis of monthly sales trends, evaluation of regional and category performance, identification of top sub-categories, and examination of how discounts affect profitability.

💡Key Insights:
The analysis shows that a small number of categories and sub-categories contribute significantly to total revenue. Some regions perform better than others in terms of profit, while higher discounts often lead to reduced or negative profit. Additionally, certain products consistently generate losses and require attention.

📁 Project Structure:
The project consists of a dataset file named Superstore.csv, a Jupyter Notebook file containing the analysis, and this README file.

▶️ How to Run?
To run this project, clone the repository, open the notebook in Jupyter, and execute all cells to reproduce the analysis and visualizations.
